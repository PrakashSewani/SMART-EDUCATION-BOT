About




Not every answer is available on the internet, students need to go through books to solve their queries which consume more time than required.Smart EducationalBot is a query solver bot that provides answers to science students of standard 11th & 12th. The bot is programmed to perform pdf scraping to display the suitable output to the user.

Features

Smart EducationalBot uses bert algorithm to process the context available and filters it out to provide the best possible answer to students.

Web Scraping

In this method, APIs are used to extract information from intended sites. Web scraping is used in this chatbot as an alternative to pdf scraping.

Keyword Extraction

Keyword extraction is the first and most important step of this system. Extraction of keywords from the query helps the bot to know about the topic of interest of the user.

PDF Scraping

Pdf scraping is a method of extracting intended info from the pdf. This is achieved using yake, tokenizer, and bert. This feature provies answers to the user from textbook pdfs.
